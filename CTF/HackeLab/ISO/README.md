## Просмотр архива
Скачиваем архив и используес команду unzip

```bash
unzip iso iso_images.zip
```

## Получаем образ диска Alpine Linux 

```bash
file alpine.iso
```
Вывод: ```alpine.iso: ISO 9660 CD-ROM filesystem data 'ALPINE'```

## Монтируем образ 

```bash
mount -o loop alpine.iso ISO/ (ISO - это заранее созданная папка куда монтироваться)
```
Используем флаг ```-o loop``` указывает системе, что монтируемый образ должен быть обработан
как блочное устройство. Это важно, потому что ISO - файл не является физическим
устройством (например, CD или DVD ), а всего лишь образом, содержащим данные.
Чтобы Linux мог работать с этим образом, ему нужно создать виртуальное блочное
устройство, и именно для этого используется loopback - устройство.
