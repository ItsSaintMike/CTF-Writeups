Используя pdf2john извлекаем хеш
Вывод:```flag.jpg:$pdf$2*3*128*-4*1*16*8f60187c054e0f37214505597f60c66c*32*46836cbd0c5fb6a9dd452481954ee90428bf4e5e4e758a4164004e56fffa0108*32*0c34c06e3dc37eb2362a2f87cf65e65551a6eb55c7306d969d7bc440f80e3193```

```pdf2john``` — это утилита, предназначенная для извлечения хэшей из зашифрованных PDF-файлов, чтобы затем можно было подобрать пароль к документу с помощью брутфорс-инструментов, таких как John the Ripper.

После полученя флага брутим наш хеш

```john --wordlist=/usr/shade/wordlist/rockyou.txt hash (наш файл с хешем)```
